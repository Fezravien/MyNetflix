# MyNetflix
Making it like Netflix 

## Features
- Storyborad
- AVFoundation(AVPlayer)
- URLComponents
- Codable
- Itunes API
- Kingfisher(Swift Package Dependencise)
- Alamofire(Swift Package Dependencise)
- Firebase(Cocoapods)
